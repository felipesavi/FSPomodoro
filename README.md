# ⏱️ FSPomodoro App
FSPomodoro es una herramienta de productividad móvil desarrollada con React Native. Esta aplicación implementa la técnica Pomodoro, ayudando a los usuarios a alternar entre periodos de enfoque y descansos cortos para mejorar la concentración y el rendimiento.

🚀 Características
Temporizador Inteligente: Configuración de tiempos para Enfoque, Descanso Corto y Descanso Largo.

Gestión de Estados: Cambio dinámico de colores y fondos según el modo seleccionado.

Interfaz Minimalista: Diseño limpio enfocado en la usabilidad y la reducción de distracciones.

Optimizado para Android: Configuración lista para dispositivos y emuladores Android.

🛠️ Tecnologías y Herramientas
React Native (Core Components)

JavaScript / ES6+

Hooks: Uso de useState y useEffect para la lógica del contador.

NPM: Gestión de dependencias.

📦 Instalación y Configuración
Para probar la aplicación en tu entorno local, sigue estos pasos:

Clonar el repositorio:

Bash
git clone https://github.com/felipesavi/FSPomodoro.git
cd FSPomodoro
Instalar las dependencias:

Bash
npm install
Ejecutar en Android:
(Asegúrate de tener un emulador abierto o tu dispositivo conectado por USB)

Bash
npx react-native run-android
📁 Estructura del Código
App.js: Contiene la lógica principal del cronómetro y el renderizado de la interfaz.

Componentes: Organización modular para los botones de control y la visualización del tiempo.

Estilos: Implementación de StyleSheet para una apariencia nativa consistente.
